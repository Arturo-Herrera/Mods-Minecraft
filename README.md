# 🎮 Mods Minecraft Java 1.20.1 — Forge

Repositorio de mods para nuestro servidor de Minecraft (Java Edition, versión **1.20.1**, con **Forge**).

---

## 📥 Instalación

> ⚠️ **Importante:** No necesitas copiar todo el repositorio.

Solo copia en tu carpeta `mods` de la laptop:

- ✅ Los mods que están **fuera de las carpetas**
- ✅ Los mods que están dentro de la carpeta **`cliente`**

No copies:

- ❌ Los mods que están en la carpeta **`servidor`** (esos son solo para el servidor, no los necesitas)

---

## 📂 Resumen rápido

| Ubicación | ¿Se copia? |
|---|---|
| Archivos sueltos (fuera de carpetas) | ✅ Sí |
| Carpeta `cliente/` | ✅ Sí |
| Carpeta `servidor/` | ❌ No |

---

¡Y listo! Con eso tu cliente debería quedar sincronizado correctamente con el servidor. 🚀
